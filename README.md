# Geeknote

yet another evernote command line client

## Installation

Add this line to your application's Gemfile:

    gem 'geeknote'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install geeknote

## Usage

    geeknote abc 
    #=> abc
    :)
